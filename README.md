This repository contains my Python code samples for Module 3. Each file corresponds to one of the seven lab problems.
Hello World - This program prints a basic message to the screen to show that Python is running correctly.
Greet User - This program asks the user for their name and then greets them using the name they typed.
Greet Specific People - This program only greets me and my instructor. If anyone else types their name, it gives a different message.
Circle Area Calculator - This program asks the user for a radius and calculates the area of a circle using that number.
MPG Calculator - This program asks for miles driven and gallons used, then calculates how many miles per gallon the car gets.
Fahrenheit to Celsius Converter - This program takes a temperature in Fahrenheit and converts it to Celsius.
Return Day Calculator - This program asks for a starting day number and the number of nights for a trip, then tells the user what day of the week they will return on.
